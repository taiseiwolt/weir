プライバシーポリシー

最終更新日: 2026年3月15日

AIden（以下「当サービス」）は、利用者の個人情報の保護を重要な責務と考え、個人情報の保護に関する法律（以下「個人情報保護法」）その他の関連法令を遵守し、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報保護法第2条第1項に定義される個人情報、すなわち、生存する個人に関する情報であって、氏名、生年月日その他の記述等により特定の個人を識別できるもの（他の情報と容易に照合でき、それにより特定の個人を識別できるものを含みます）をいいます。

第3条（個人情報の利用目的）

  1. 注文の処理、配達、および関連する連絡
  2. アカウントの作成・認証・管理
  3. ポイントプログラムおよび会員ランクの管理
  4. お問い合わせへの対応
  5. サービスの改善および新機能の開発
  6. 利用規約違反への対応
  7. 利用状況の統計・分析（個人を特定しない形式）
  8. マーケティングおよびキャンペーンに関するご案内（利用者の同意がある場合に限る）

第4条（利用目的の変更）

  1. 当サービスは、利用目的を相当の関連性を有すると合理的に認められる範囲内で変更することがあります。
  2. 利用目的を変更した場合、変更後の目的を利用者に通知または本サービス上で公表します。

第5条（個人情報の第三者提供）

  1. 当サービスは、以下の場合を除き、利用者の個人情報を第三者に提供しません。
    1. 利用者の同意がある場合
    2. 法令に基づく場合
    3. 人の生命、身体または財産の保護のために必要な場合であって、利用者の同意を得ることが困難であるとき
    4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
    5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行するために協力する必要がある場合
  2. 注文処理のため、加盟店に対して注文に必要な最小限の情報（注文内容、注文タイプ等）を提供します。
  3. ゲスト注文の場合、利用者の個人情報（氏名、メールアドレス、電話番号等）は加盟店に共有されません。加盟店に共有されるのは、注文件数と注文日時のみです。
  4. 利用目的の達成に必要な範囲で、個人情報の取り扱いを外部に委託する場合があります。その際、委託先に対して適切な監督を行います。

第7条（外部送信に関する事項）

当サービスは、サービスの利便性向上および利用状況の分析のため、利用者の端末に保存された情報を外部に送信する場合があります（電気通信事業法第27条の12に基づく開示）。

利用者は、ブラウザの設定によりCookieの受け入れを拒否することができますが、一部の機能が制限される場合があります。

第8条（クレジットカード情報の取り扱い）

  1. 当サービスは、クレジットカード情報を直接保持しません。
  2. カード情報の入力・保存・処理はすべてStripe, Inc.のシステム上で行われ、PCI DSSに準拠した環境で管理されます。
  3. 会員がカード情報を保存する場合、Stripe上の「Customer」機能を通じて保管されます。

第9条（口コミ・評価の取り扱い）

  1. 会員が投稿した口コミ・評価（テキスト、写真等）は、投稿者に帰属します。
  2. 投稿者は当サービスに対し、口コミ・評価を本サービスの提供に必要な範囲で利用（表示、加工、分析等を含む）する権利を無償で許諾するものとします。
  3. 口コミ・評価は、他の利用者に公開されます。投稿者の氏名は表示されず、ニックネームまたは匿名で表示されます。
  4. 当サービスは、利用規約に違反する口コミ・評価を予告なく削除することがあります。

第10条（個人情報の安全管理措置）

当サービスは、個人情報の漏洩、滅失、毀損の防止のため、以下の安全管理措置を講じます。

  1. 組織的安全管理措置：個人情報の取り扱いに関する責任者を設置し、取り扱う個人情報の範囲を明確化しています。
  2. 人的安全管理措置：個人情報の取り扱いに関する留意事項について、定期的な確認を実施しています。
  3. 技術的安全管理措置：パスワードはハッシュ化して保存し、平文での保存は行いません。通信はSSL/TLSにより暗号化されます。アクセス制御により、権限のない者による個人情報へのアクセスを防止しています。
  4. 物理的安全管理措置：個人情報はクラウド環境（Supabase）に保管されており、適切なアクセス制御が施されています。

第12条（利用者の権利）

利用者は、個人情報保護法に基づき、当サービスに対して以下の請求を行うことができます。

  1. 利用目的の通知の請求
  2. 個人情報の開示の請求
  3. 個人情報の訂正、追加または削除の請求
  4. 個人情報の利用停止または消去の請求
  5. 個人情報の第三者提供の停止の請求
  6. 第三者提供記録の開示の請求

請求手続き：

  1. [email protected] まで、氏名および登録メールアドレスを記載の上ご連絡ください。
  2. 本人確認の上、合理的な期間内（原則として2週間以内）に対応いたします。
  3. 代理人による請求の場合は、委任状および代理人の本人確認書類が必要です。

第13条（Cookieおよびアクセスログ）

  1. 当サービスは、サービスの利便性向上のためCookieを使用する場合があります。
  2. Cookieにより個人を特定する情報は収集しません。
  3. ブラウザの設定によりCookieを無効にすることができますが、一部機能が制限される場合があります。
  4. 当サービスは、サービスの運営・改善のため、アクセスログ（IPアドレス、ブラウザの種類、アクセス日時等）を自動的に収集します。

第14条（未成年者の利用）

  1. 未成年者が本サービスを利用する場合、法定代理人の同意を得た上でご利用ください。
  2. 当サービスは、16歳未満の方から故意に個人情報を収集しません。

第15条（個人情報の漏洩等への対応）

  1. 万が一、個人情報の漏洩、滅失、毀損等が発生した場合、速やかに原因を調査し、被害の拡大防止に努めます。
  2. 個人情報保護法に基づき、個人情報保護委員会への報告および本人への通知を行います。

第16条（ポリシーの変更）

  1. 当サービスは、必要に応じて本ポリシーを変更できます。
  2. 重要な変更がある場合は、本サービス上またはメールにて事前に通知します。
  3. 変更後も本サービスを利用した場合、変更後のポリシーに同意したものとみなします。
