# AIden プライバシーポリシー

**ドラフト v1.0 | 2026年3月24日**

---

## 第1条（基本方針）

AIden株式会社（以下「当社」という）は、モバイルオーダーサービス「AIden」（以下「本サービス」という）の利用者の個人情報を、本ポリシーに従い取り扱う。当社は、個人情報の保護に関する法律（個人情報保護法）その他の関連法令を遵守する。

## 第2条（個人情報管理者）

個人情報管理に関する責任者は、以下のとおりとする。

- 事業者名: AIden株式会社
- 所在地: （確定後記載）
- 個人情報保護管理者: （確定後記載）
- 連絡先: support@aiden-jp.net

## 第3条（収集する個人情報の種類）

### 3-1. 会員から収集する情報

| 情報項目 | 収集タイミング | 必須/任意 |
|---------|-------------|---------|
| 氏名 | 会員登録時 | 必須 |
| メールアドレス | 会員登録時 | 必須 |
| 電話番号 | 会員登録時 | 必須 |
| 配達先住所 | デリバリー注文時 | デリバリー利用時必須 |
| 注文履歴 | 注文時 | 自動取得 |
| ポイント残高・履歴 | ポイント付与・利用時 | 自動取得 |
| クーポン利用履歴 | クーポン利用時 | 自動取得 |
| LINE アカウント情報 | LINE連携時 | LINE連携利用時必須 |
| 決済カード情報 | 注文時 | 必須（Stripe経由で処理。当社サーバーには保存しない） |

### 3-2. ゲスト利用者から収集する情報

| 情報項目 | 収集タイミング | 必須/任意 |
|---------|-------------|---------|
| 氏名 | ゲスト注文時 | 必須 |
| メールアドレス | ゲスト注文時 | 必須 |
| 決済カード情報 | 注文時 | 必須（Stripe経由で処理。当社サーバーには保存しない） |

### 3-3. 自動取得する情報

| 情報項目 | 取得方法 |
|---------|--------|
| アクセスログ（IPアドレス、ブラウザ種別、アクセス日時） | Webサーバーのログ |
| 端末情報（OS種別、画面サイズ） | ブラウザから自動取得 |
| Cookie情報 | ブラウザのCookie |

### 3-4. 音声データ

| 情報項目 | 保持期間 |
|---------|--------|
| 音声コマンドデータ（"Hey AIden" ウェイクワード起動後の音声） | 90日間。90日経過後に自動削除する |

## 第4条（利用目的）

当社は、収集した個人情報を以下の目的で利用する。これらの目的以外には利用しない。

1. 注文の処理および配達の実施
2. 決済処理（Stripe, Inc.を通じたクレジットカード決済）
3. ポイントの付与、管理および利用処理
4. クーポンの発行および利用管理
5. メンバーシッププログラムの運営
6. カスタマーサポートへの対応（チャットサポート含む）
7. AI機能によるコンテンツ生成（レビュー返信、SNS投稿文案、POP画像生成）への利用。AI生成コンテンツに利用者個人を特定する情報は含めない
8. サービス利用状況の分析および本サービスの改善
9. 本サービスに関する通知（注文ステータス、規約変更等）の送信
10. 不正利用の検知および防止
11. 法令に基づく対応

## 第5条（第三者提供）

### 5-1. 加盟店への情報提供

当社は、注文の履行に必要な範囲で、以下の情報を加盟店に提供する。

**会員の場合:**
- 注文内容（商品名、数量、金額）
- 注文日時
- 注文種別（イートイン / テイクアウト / デリバリー）
- デリバリーの場合: 配達先住所、電話番号
- 注文回数

**ゲスト利用者の場合:**
- 注文内容（商品名、数量、金額）
- 注文日時
- 注文種別
- デリバリーの場合: 配達先住所（注文処理目的に限定）
- 注文回数

**ゲスト利用者の氏名およびメールアドレスは、加盟店に提供しない。** 加盟店が閲覧できるのは、注文回数および注文日時のみである。

### 5-2. 決済事業者への情報提供

決済処理に必要な情報を、Stripe, Inc.に提供する。Stripe, Inc.による個人情報の取扱いは、同社のプライバシーポリシーに従う。

### 5-3. その他の第三者提供

当社は、以下の場合を除き、利用者の個人情報を第三者に提供しない。

1. 利用者本人の同意がある場合
2. 法令に基づく場合（裁判所の命令、捜査機関からの照会等）
3. 人の生命、身体または財産の保護に必要な場合で、利用者本人の同意を得ることが困難な場合

## 第6条（AI機能における個人情報の取扱い）

1. 当社のAI機能（Claude API、OpenAI API）は、加盟店向けのコンテンツ生成（レビュー返信、SNS投稿文案、POP画像等）に使用する。
2. AI機能への入力データに、利用者個人を特定する情報（氏名、メールアドレス、電話番号等）は含めない。
3. AI機能が生成したコンテンツに、利用者個人を特定する情報が含まれていないことを確認した上で公開する。
4. AI生成コンテンツの学習データとして、利用者の個人情報は使用しない。

## 第7条（安全管理措置）

当社は、個人情報の漏洩、滅失、毀損を防止するため、以下の安全管理措置を講じる。

1. **技術的措置**
   - データベースにRow Level Security（RLS）を適用し、アクセス権限を制御する
   - 通信の暗号化（TLS/SSL）を実施する
   - 認証にはJWT（JSON Web Token）を使用し、適切な有効期限を設定する
   - カード情報は当社サーバーに保存せず、Stripe, Inc.のPCI DSS準拠環境で処理する
2. **組織的措置**
   - 個人情報へのアクセス権限を必要最小限の者に限定する
   - サービスロールキー（管理者権限）はサーバーサイドのみで使用し、フロントエンドに露出させない

## 第8条（Cookie・アクセスログの取扱い）

1. 当社は、本サービスの利便性向上および利用状況の分析を目的として、Cookieを使用する。
2. Cookieにより収集する情報は以下のとおりである。
   - セッション情報（ログイン状態の維持）
   - 利用者の設定情報（言語、表示設定等）
3. 利用者は、ブラウザの設定によりCookieの受け入れを拒否することができる。ただし、Cookieを拒否した場合、本サービスの一部機能が利用できなくなる場合がある。
4. アクセスログ（IPアドレス、ブラウザ種別、アクセス日時）は、不正アクセスの検知およびサービス改善の目的で、12か月間保持する。

## 第9条（データ保持期間）

| データ種別 | 保持期間 |
|-----------|--------|
| 会員情報 | 退会後90日間（復元申請期間30日＋削除処理期間60日） |
| ゲスト注文に伴う個人情報 | 注文日から90日間 |
| 注文履歴 | 無期限（個人情報を削除した上で匿名化して保持） |
| ポイント・クーポン履歴 | 会員退会後90日間 |
| チャットサポート履歴 | 180日間 |
| 音声データ | 90日間 |
| アクセスログ | 12か月間 |

## 第10条（開示・訂正・削除請求）

1. 利用者は、当社に対し、自己の個人情報の開示、訂正、削除を請求することができる。
2. 請求方法は、以下のとおりとする。
   - メール: support@aiden-jp.net 宛に、件名「個人情報開示請求」「個人情報訂正請求」「個人情報削除請求」のいずれかを記載し、本人確認書類を添付して送信する
3. 当社は、請求を受領した日から14日以内に対応を完了し、結果を請求者に通知する。
4. 本人確認ができない場合、請求に応じることはできない。

## 第11条（個人情報漏洩時の対応）

1. 当社は、個人情報の漏洩が発生した場合、以下の対応を行う。
   - 漏洩を認知した時点から24時間以内に、影響を受ける利用者にメールで通知する
   - 個人情報保護委員会に報告する（個人情報保護法第26条に基づく）
   - 漏洩の原因を特定し、再発防止措置を講じる
   - 漏洩の経緯、影響範囲、再発防止策を公表する
2. 漏洩の影響を最小限にするため、対象データのアクセス権限を即時制限する。

## 第12条（退会時の個人情報の取扱い）

1. 会員が退会した場合、以下の処理を行う。
   - 退会後30日間: アカウント復元申請を受け付ける期間。個人情報は保持する
   - 退会後31日〜90日: 個人情報の削除処理を実施する。注文履歴は customer_id をNULL化し匿名化する
   - 退会後91日以降: 個人情報は完全に削除される。匿名化された注文履歴のみ残存する
2. チャットサポート履歴の customer_id は退会時にNULL化する。

## 第13条（ポリシーの変更）

1. 当社は、法令の改正、サービス内容の変更、その他の事由により、本ポリシーを変更する場合がある。
2. 変更する場合は、変更の効力発生日の30日前までに、本サービス上での掲示および登録メールアドレスへの通知により利用者に周知する。

## 第14条（問い合わせ先）

本ポリシーに関する問い合わせは、以下の窓口で受け付ける。

- メール: support@aiden-jp.net
- 受付時間: 平日10:00〜18:00（土日祝日・年末年始を除く）
