Weir エンドユーザー利用規約

バージョン 1.0 ｜ 施行日: 2026年4月1日

第1条（定義）

本規約において、以下の用語は次の意味で使用する。

  1. 「本サービス」とは、Weir株式会社（以下「当社」という）が提供する飲食店向けプラットフォーム「Weir」を通じて、エンドユーザーが利用できるモバイルオーダー、ポイントプログラム、クーポン等の各種サービスの総称をいう。
  2. 「エンドユーザー」とは、本サービスを通じて飲食店に注文を行う消費者をいう。
  3. 「事業者」とは、本サービスを利用して飲食店を運営する法人または個人事業主をいう。
  4. 「会員」とは、本規約に同意の上、所定の手続きにより会員登録を完了したエンドユーザーをいう。
  5. 「ゲスト」とは、会員登録を行わずに本サービスを利用するエンドユーザーをいう。
  6. 「ポイント」とは、本サービスにおいて会員に付与される特典ポイントをいう。
  7. 「クーポン」とは、事業者が発行する割引等の特典をいう。

第2条（サービス内容）

  1. 本サービスは、以下の機能を含む。
    • モバイルオーダー（店内飲食、テイクアウト、デリバリー）
    • ポイントプログラム（別途「ポイント利用規約」に定める）
    • クーポン（別途「クーポン規約」に定める）
    • 注文履歴の閲覧
    • その他当社が提供する付随サービス
  2. 本サービスの内容は、当社の判断により変更、追加または廃止されることがある。

第3条（会員登録・認証）

  1. 会員登録を希望するエンドユーザーは、当社所定の方法により登録手続きを行うものとする。
  2. 会員登録にはメールアドレスの認証が必要であり、認証コードの有効期限は発行から60分以内とする。
  3. エンドユーザーは、登録情報が正確かつ最新であることを保証するものとする。
  4. 以下に該当する場合、当社は会員登録を拒否し、または会員資格を取り消すことができる。
    • 登録情報に虚偽があった場合
    • 過去に本規約に違反したことがある場合
    • その他当社が不適切と判断した場合

第4条（注文・決済）

  1. エンドユーザーは、本サービスを通じて事業者の提供する商品を注文することができる。
  2. 決済は、Stripe株式会社（以下「Stripe」という）の決済システムを通じて処理される。
  3. 注文時にクレジットカード等の決済手段に対して与信枠の確保（オーソリゼーション）が行われ、商品の提供完了時に売上確定（キャプチャ）が行われる。
  4. 注文の成立は、当社システムにおいて注文が受け付けられた時点とする。ただし、事業者が注文を拒否した場合は、注文は不成立となり、オーソリゼーションは取り消される。
  5. 商品の価格、内容、提供時間等は事業者が設定するものであり、当社はこれらについて責任を負わない。

第5条（キャンセル・返金）

  1. 注文確定前（オーソリゼーション段階）のキャンセルについては、オーソリゼーションの取消が行われ、返金処理は発生しない。
  2. 注文確定後の返金については、以下の条件に従う。
    • プラットフォーム側の返金期限は、注文日から90日以内とする。
    • 各店舗が独自に設定する返金期限が適用される場合がある。
  3. 返金はStripeを通じて、元の決済手段に対して行われる。
  4. 返金の詳細は、別途「返金ポリシー」に定める。

第6条（ポイント）

  1. 会員は、注文完了時に事業者が設定するポイントの付与を受けることができる。
  2. ポイントの付与条件、利用条件、有効期限等の詳細は、別途「ポイント利用規約」に定める。
  3. 退会時には、保有するすべてのポイントが即時に失効する。なお、退会手続き時にポイント失効に関する通知が行われる。
  4. ポイントは現金への換金、他サービスへの移行等はできない。

第7条（クーポン）

  1. クーポンは事業者が発行するものであり、当社はクーポンの仕組みを提供する立場にある。
  2. クーポンの使用条件（有効期限、最低注文金額、対象商品等）は、事業者が設定する。
  3. 退会時には、保有するすべてのクーポンが無効となる。
  4. クーポンの詳細は、別途「クーポン規約」に定める。

第8条（禁止事項）

エンドユーザーは、本サービスの利用にあたり、以下の行為を行ってはならない。

  1. 法令または公序良俗に違反する行為
  2. 虚偽の情報を登録または提供する行為
  3. 他人の個人情報、アカウント情報等を不正に使用する行為
  4. 不正な手段によりポイントまたはクーポンを取得する行為
  5. 本サービスのシステムに対する不正アクセス、改ざん、妨害行為
  6. 事業者または他のエンドユーザーに対する迷惑行為、脅迫、誹謗中傷
  7. 本サービスを営利目的で利用する行為（転売等）
  8. その他当社が不適切と判断する行為

第9条（退会）

  1. 会員は、本サービスの所定の手続きにより、いつでも退会することができる。退会は即時に効力を生じる。
  2. 退会時に以下の処理が行われる。
    • 保有するすべてのポイントの即時失効
    • 保有するすべてのクーポンの無効化
    • 会員としてのサービス利用権の喪失
  3. 退会後、当社はエンドユーザーの個人情報を退会日から90日以内に匿名化する。ただし、注文履歴は税法上の義務に基づき注文日から7年間保持する。
  4. 退会後の再登録は、新規登録として扱われ、失効したポイントおよびクーポンは復元されない。

第10条（免責事項）

  1. 当社は、本サービスの完全性、正確性、有用性、特定目的への適合性等について保証しない。
  2. 当社は、事業者が提供する商品の品質、安全性、適法性等について責任を負わない。
  3. 当社は、天災地変、システム障害、通信障害その他の不可抗力に起因する本サービスの中断、遅延、データの消失等について責任を負わない。
  4. 当社は、エンドユーザーと事業者との間のトラブルについて、原則として関与しない。ただし、当社が必要と認める場合は、仲介その他の措置を講じることができる。
  5. 当社の責任は、当社に故意または重過失がある場合を除き、エンドユーザーが被った直接かつ現実の損害に限られ、間接損害、逸失利益等については責任を負わない。

第11条（個人情報の取扱い）

  1. 当社は、エンドユーザーの個人情報を、別途定める「プライバシーポリシー」に従い適切に取り扱う。
  2. エンドユーザーは、本サービスの利用にあたり、プライバシーポリシーに同意するものとする。

第12条（規約の変更）

  1. 当社は、本規約を変更する場合、変更後の規約の内容および施行日を、施行日の30日前までに本サービス上で通知する。
  2. 変更後の規約の施行日以降に本サービスを利用した場合、当該エンドユーザーは変更後の規約に同意したものとみなす。
  3. 重要な変更の場合、当社は登録されたメールアドレスに対しても通知を行うよう努める。

第13条（準拠法・管轄）

  1. 本規約の解釈および適用は、日本法に準拠する。
  2. 本規約に関連する一切の紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とする。

附則
本規約は2026年4月1日より施行する。
