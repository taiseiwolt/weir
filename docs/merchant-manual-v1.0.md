## 1. はじめに

### 1.1 AIden でできること

AIden は飲食店の運営をまとめてサポートするサービスです。主に以下のことができます。

| 機能 | 概要 |
|------|------|
| **モバイルオーダー** | お客様がスマートフォンから注文・決済できます |
| **受注ダッシュボード** | 入った注文をリアルタイムで確認・管理できます |
| **ブランドHP** | お店のホームページを自動で作成・公開できます |
| **顧客管理** | 会員のお客様の情報や来店履歴を管理できます |
| **AI支援** | レビュー返信やSNS投稿文をAIが自動生成します |
| **メンバーシップ** | ポイントプログラムやクーポンでリピーターを増やせます |

### 1.2 使う画面の一覧

AIden では主に3つの画面を使います。

| 画面名 | 用途 | いつ使う？ |
|--------|------|-----------|
| **受注ダッシュボード** | 注文の確認・ステータス管理・返金 | 毎日（営業中は常時表示推奨） |
| **管理マスタ** | メニュー・営業時間・店舗情報の設定変更 | 設定を変更したい時 |
| **顧客管理画面** | 会員情報の確認・クーポン管理 | 顧客データを確認したい時 |

### 1.3 推奨環境

- **ブラウザ:** Google Chrome（最新版）または Safari（最新版）
- **受注ダッシュボード:** タブレット推奨（キッチンに設置して常時表示）
- **管理画面:** PC またはタブレット

## 2. 受注ダッシュボード（日常操作）

受注ダッシュボードは、注文の受付から完了まで管理する画面です。営業中は常にこの画面を開いておいてください。

### 2.1 ログイン方法

1. ブラウザ（Chrome や Safari）を開きます
2. アドレスバー（画面上部のURL入力欄）に `https://aiden-jp.net/aiden-order-dashboard.html` と入力します
3. **メールアドレス**と**パスワード**を入力します
4. 「ログイン」ボタンをタップします

※ ログイン情報は導入時にお渡しします。パスワードを忘れた場合はサポートにご連絡ください。

---

### 2.2 新着注文の確認

新しい注文が入ると、以下のように通知されます。

- 画面上に**新着注文カード**が表示されます
- **通知音**が鳴ります

**注文カードに表示される情報：**
- 注文番号
- 注文タイプ（イートイン / テイクアウト / デリバリー）
- 注文された商品と数量
- 合計金額
- お客様の受取希望時間（設定されている場合）

---

### 2.3 ステータス（注文の進み具合）の変更

注文には4つのステータスがあります。順番に変更していきます。

| ステータス | 意味 | いつ変更する？ |
|------------|------|---------------|
| **新着** | 注文が入ったばかり | — |
| **受付済み** | 注文を確認し、対応を開始 | 注文内容を確認した時 |
| **準備中** | 料理を準備している最中 | 調理を開始した時 |
| **完了** | 料理の提供が完了 | お客様にお渡しした時 |

**ステータスを変更する手順：**

1. 変更したい注文カードをタップします
2. 注文の詳細画面が表示されます
3. 画面下部の**ステータス変更ボタン**をタップします
4. 次のステータスに切り替わります

※ ステータスを変更すると、お客様のスマートフォン画面にもリアルタイムで反映されます。こまめに更新するとお客様の安心感が高まります。

---

### 2.4 返金操作

注文のキャンセルや間違いがあった場合に返金を行います。

**返金の手順：**

1. 受注ダッシュボードで対象の注文カードをタップします
2. 注文詳細画面で「**返金**」ボタンをタップします
3. 返金タイプを選択します
   - **全額返金**: 注文金額のすべてを返金
   - **一部返金**: 金額を指定して一部だけ返金
4. 一部返金の場合、返金額を入力します
5. 「**返金を実行**」ボタンをタップします
6. 確認画面で「**確定**」をタップします

**注意事項：**
- 返金は注文日から**90日以内**に行ってください（期間は管理画面で変更可能）
- 返金が完了するとお客様のカード会社経由で返金されます（反映まで数日かかる場合があります）

---

### 2.5 売上サマリの見方

受注ダッシュボードには売上の概要が表示されます。

**確認できる情報：**
- 本日の注文件数
- 本日の売上合計
- 注文タイプ別の内訳（イートイン / テイクアウト / デリバリー）

※ より詳しい売上分析は、管理マスタ画面から確認できます。

## 3. 管理マスタ（設定変更）

管理マスタは、メニューや営業時間などお店の設定を変更する画面です。

### 3.1 管理マスタへのアクセス

1. ブラウザで `https://aiden-jp.net/aiden-admin.html` を開きます
2. ログインします（受注ダッシュボードと同じアカウント）

---

### 3.2 メニューの追加・編集・非公開

#### 新しいメニューを追加する

1. 管理マスタ画面で「**メニュー管理**」を選択します
2. 「**新規追加**」ボタンをタップします
3. 以下の情報を入力します
   - **商品名**: メニュー名（例: 特上にぎり盛り合わせ）
   - **価格**: 税込価格（例: 1,980）
   - **カテゴリ**: 分類（例: にぎり、丼もの、サイドメニュー）
   - **説明文**: 商品の説明（お客様に表示されます）
   - **写真**: 商品の写真をアップロード
4. 「**保存**」ボタンをタップします

#### メニューを編集する

1. メニュー一覧から編集したい商品をタップします
2. 変更したい項目を修正します
3. 「**保存**」ボタンをタップします

#### メニューを非公開にする（一時的に注文を受けたくない場合）

1. メニュー一覧から該当する商品をタップします
2. 「**公開設定**」を「非公開」に切り替えます
3. 「**保存**」ボタンをタップします

※ 非公開にすると、お客様の注文画面にはその商品が表示されなくなります。再度「公開」に戻せばいつでも復活します。

---

### 3.3 営業時間の変更

1. 管理マスタ画面で「**店舗設定**」を選択します
2. 「**営業時間**」セクションを開きます
3. 曜日ごとに**開店時間**と**閉店時間**を設定します
4. 定休日がある場合は該当曜日を「**休業**」に設定します
5. 「**保存**」ボタンをタップします

※ 臨時休業や特別営業日の設定もこの画面から行えます。

---

### 3.4 配達設定

デリバリーを行う場合の設定です。

1. 管理マスタ画面で「**配達設定**」を選択します
2. 以下の項目を設定します
   - **配達対応**: オン / オフ
   - **配達エリア**: 配達可能な範囲
   - **配達料金**: お客様にかかる配達料（無料にもできます）
   - **最低注文金額**: 配達を受ける最低金額
   - **配達予想時間**: 配達にかかる目安時間
3. 「**保存**」ボタンをタップします

---

### 3.5 店舗情報の編集

1. 管理マスタ画面で「**店舗設定**」を選択します
2. 以下の項目を編集できます
   - 店舗名
   - 住所
   - 電話番号
   - 店舗の紹介文
   - ロゴ画像
   - ヒーロー画像（ホームページのトップに表示される大きな画像）
3. 「**保存**」ボタンをタップします

※ ブランドHP にも反映されます。

## 4. 顧客管理画面

顧客管理画面では、会員登録したお客様の情報やクーポンを管理できます。

### 4.1 顧客管理画面へのアクセス

1. ブラウザで `https://aiden-jp.net/aiden-customer-admin.html` を開きます
2. ログインします

---

### 4.2 顧客一覧の見方

顧客一覧画面では以下の情報を確認できます。

| 項目 | 内容 |
|------|------|
| 会員番号 | お客様のID |
| 会員ランク | ポイントに応じたランク |
| 注文回数 | 累計の注文回数 |
| 累計利用金額 | 累計の注文金額 |
| 最終注文日 | 最後に注文した日 |
| ポイント残高 | 現在のポイント数 |

**検索・フィルター機能：**
- 名前やメールアドレスで検索
- ランクで絞り込み
- 最終注文日で並べ替え

※ ゲスト注文（会員登録していないお客様）の場合、個人情報は表示されません。注文回数と日時のみ確認できます。

---

### 4.3 クーポンの作成・管理

#### 新しいクーポンを作成する

1. 顧客管理画面で「**クーポン管理**」を選択します
2. 「**新規クーポン作成**」ボタンをタップします
3. 以下の項目を設定します
   - **クーポン名**: 表示名（例: 初回来店10%OFF）
   - **割引タイプ**: 割合（%OFF）または 固定額（〇〇円引き）
   - **割引値**: 割引の数値（例: 10% または 500円）
   - **有効期限**: クーポンの使用期限
   - **利用条件**: 最低注文金額など（任意）
   - **対象者**: 全員 / 特定ランクの会員 / 新規会員 など
4. 「**作成**」ボタンをタップします

#### クーポンを無効にする

1. クーポン一覧から該当のクーポンを選択します
2. 「**無効にする**」をタップします
3. 確認画面で「**確定**」をタップします

## 5. AI機能の使い方

### 5.1 レビュー返信の生成・確認・公開

Googleレビューに対する返信文をAIが自動生成します。

**操作手順：**

1. 管理マスタ画面で「**レビュー管理**」を選択します
2. 返信が必要なレビューの横にある「**AI返信を生成**」ボタンをタップします
3. AIが返信文を生成します（数秒〜十数秒かかります）
4. **プレビュー画面**で内容を確認します
5. 必要に応じて**文面を編集**します
6. 内容に問題がなければ「**公開**」ボタンをタップします

**ポイント：**
- AIは丁寧で好印象な返信文を生成しますが、事実と異なる内容が含まれていないか必ず確認してください
- 低評価レビューへの返信は特に慎重にご確認ください
- STDプランでは月10件まで無料で生成できます

---

### 5.2 SNS投稿の生成・確認・投稿

X（旧Twitter）やInstagramへの投稿文をAIが自動生成します。

**操作手順：**

1. 管理マスタ画面で「**SNS管理**」を選択します
2. 投稿先のSNSを選択します（X / Instagram）
3. 投稿のテーマを選択または入力します（例: 本日のおすすめ、新メニュー紹介）
4. 「**AI投稿文を生成**」ボタンをタップします
5. AIが投稿文を生成します
6. **プレビュー画面**で内容を確認します
7. 必要に応じて**文面を編集**します
8. 「**投稿**」ボタンをタップします

**ポイント：**
- Xは自動投稿が可能です
- Instagramは現在、生成された文面をコピーして手動で投稿していただく形になります（今後自動投稿に対応予定）
- STDプランでは月10件まで無料で生成できます

---

### 5.3 POP画像の生成

メニューのPOP画像（販促用の画像）をAIが自動生成します。

**操作手順：**

1. 管理マスタ画面で「**POP画像生成**」を選択します
2. 対象の商品を選択します
3. 画像のスタイルを選択します（例: ポップ、シンプル、高級感 など）
4. 「**画像を生成**」ボタンをタップします
5. AIが画像を生成します（数十秒かかる場合があります）
6. 生成された画像を確認します
7. 気に入った画像があれば「**保存**」をタップします

※ STDプランでは月1件まで無料で生成できます。

## 6. よくある操作

### 6.1 プラン変更の方法

1. 管理マスタ画面にログインします
2. 「**アカウント設定**」を選択します
3. 「**プラン変更**」をタップします
4. 変更先のプランを選択します
5. 内容を確認して「**変更を申請**」をタップします

**アップグレード（上位プランへの変更）の場合：**
- AIden オペレーターの承認後、即時反映されます
- 当月分は日割り計算で差額をお支払いいただきます

**ダウングレード（下位プランへの変更）の場合：**
- 翌月1日から新しいプランが適用されます
- 当月中は現在のプランの機能がそのままお使いいただけます

### 6.3 緊急時の連絡先

以下のような緊急事態が発生した場合は、すぐにご連絡ください。

- **注文を受け付けられない**（システム障害の可能性）
- **決済に関するトラブル**（二重請求など）
- **お客様の個人情報に関する問題**

**緊急連絡の手順：**

1. まず担当者の携帯電話にご連絡ください（番号はオンボーディング時にお伝えします）
2. 電話がつながらない場合は、メール（support@aiden-jp.net）に「**【緊急】**」とつけてご連絡ください
3. メールには以下の情報を記載してください
   - 店舗名
   - 発生している問題の内容
   - 発生時刻
   - 影響を受けている注文がある場合はその注文番号
