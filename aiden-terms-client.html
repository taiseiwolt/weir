加盟店向けサービス利用規約

最終更新日: 2026年3月15日

本規約は、AIden（以下「当サービス」）が飲食店事業者（以下「加盟店」）に対して提供する受注管理、モバイルオーダー、ブランドホームページ、顧客管理（CRM）、AI分析、およびこれらに付随するサービス（以下「本サービス」）の利用条件を定めるものです。

第1条（適用）

  1. 本規約は、加盟店と当サービスとの間の本サービスの利用に関わる一切の関係に適用されます。
  2. 加盟店は、本サービスの利用を開始した時点で本規約に同意したものとみなします。

第2条（定義）

  1. 「加盟店」とは、当サービスに加盟店登録を行い、本サービスを利用する飲食店事業者をいいます。
  2. 「エンドユーザー」とは、加盟店の商品を本サービスを通じて注文する消費者をいいます。
  3. 「管理画面」とは、加盟店が本サービスの各機能を操作するためのウェブインターフェースをいいます。
  4. 「サービスユニット」とは、本サービスを構成する個別の機能単位をいいます。

第3条（サービス内容）

  1. 本サービスは、以下のサービスユニットで構成されます。加盟店は必要なサービスユニットを選択して利用できます。
    1. モバイルオーダー（店内飲食/テイクアウト/デリバリー）
    2. デリバリーハブ
    3. AI分析（Standard / PRO / EXPERT）
    4. CRM・顧客管理（Standard / PRO / EXPERT）
  2. AI分析およびCRMのStandard機能は無料で提供されます。PRO/EXPERT機能は有料です。
  3. 各サービスユニットの提供時期・提供範囲は、当サービスの開発状況により異なる場合があります。利用可能なサービスユニットの最新情報は、管理画面上またはお問い合わせにてご確認ください。
  4. サービスユニットの選択は、法人→ブランド→店舗の階層構造に基づいて管理されます。

第4条（加盟店登録）

  1. 加盟店登録を希望する事業者は、当サービス所定の方法により申請を行うものとします。
  2. 当サービスは、申請内容を審査の上、登録の可否を判断します。
  3. 虚偽の情報による申請、反社会的勢力に該当する場合等、当サービスは登録を拒否できます。

第5条（料金と支払い）

  1. 本サービスは固定月額制ではなく、選択したサービスユニットに応じた従量課金制です。
  2. 料金体系は、別途定める料金表に従います。
  3. 決済処理はStripeを通じて行われます。売上金の精算方法は別途定めるものとします。
  4. 請求書は毎月発行され、指定の方法でお支払いいただきます。

第6条（注文処理と決済）

  1. エンドユーザーからの注文は、管理画面およびPOSシステムを通じて加盟店に通知されます。
  2. 注文受付後、加盟店は速やかに注文を確認し、調理を開始するものとします。
  3. 調理完了時に決済が確定（キャプチャ）されます。
  4. 加盟店の判断で注文をキャンセルする場合、エンドユーザーへの返金処理が自動的に行われます。

第7条（エンドユーザー情報の取り扱い）

  1. 会員登録済みエンドユーザーの情報のうち、注文処理に必要な情報は加盟店に提供されます。
  2. ゲスト注文のエンドユーザーの個人情報（氏名、メールアドレス、電話番号等）は、加盟店に共有されません。共有されるのは注文件数と注文日時のみです。
  3. 加盟店は、本サービスを通じて取得したエンドユーザー情報を、注文処理以外の目的に使用してはなりません。
  4. 加盟店は、個人情報保護法その他の関連法令を遵守し、エンドユーザー情報を適切に管理するものとします。

第8条（知的財産）

  1. 本サービスに関する知的財産権は当サービスに帰属します。
  2. 加盟店が登録したコンテンツ（メニュー画像、店舗情報等）の権利は加盟店に帰属します。
  3. 加盟店は、登録コンテンツについて、当サービスが本サービスの提供に必要な範囲で使用することを許諾します。

第9条（禁止事項）

  1. 食品衛生法その他の関連法令に違反する行為
  2. エンドユーザー情報の本サービス外での流用
  3. 本サービスのリバースエンジニアリング、不正アクセス
  4. 虚偽のメニュー情報、価格情報の掲載
  5. 当サービスの信用を毀損する行為

第10条（契約の解除）

  1. 加盟店は、1ヶ月前の書面通知により、本契約を解除できます。
  2. 当サービスは、加盟店が本規約に重大な違反をした場合、催告なく契約を解除できます。
  3. 契約解除後も、未決済の注文および未払いの料金に関する義務は存続します。

第11条（契約終了時のデータの取り扱い）

  1. 契約終了後、当サービスは加盟店のデータ（メニュー情報、店舗情報等）を30日間保持した後、削除します。
  2. 加盟店は、契約終了前または終了後30日以内に、当サービス所定の方法でデータの引き渡しを申請することができます。
  3. 注文履歴および取引記録は、税務・法令上の要件に基づき、契約終了後も最大7年間保存する場合があります。
  4. エンドユーザーの口コミ・評価は、加盟店の退店後も本サービス上に保持される場合があります。

第12条（免責事項）

  1. システム障害、天災等の不可抗力による本サービスの中断について、当サービスは責任を負いません。
  2. エンドユーザーとの間のトラブル（商品の品質、アレルギー事故等）は加盟店の責任とします。
  3. 配達に関する事故・遅延は、配達サービス提供者の責任とします。

第13条（秘密保持）

  1. 加盟店および当サービスは、本契約に関連して知り得た相手方の秘密情報を第三者に開示しないものとします。
  2. 本条の義務は、契約終了後3年間存続します。

第14条（反社会的勢力の排除）

  1. 加盟店は、自らおよびその役員・従業員が反社会的勢力（暴力団、暴力団員、暴力団準構成員、暴力団関係企業、総会屋等、その他これに準ずる者）に該当しないこと、および反社会的勢力と関係を有していないことを表明し、保証するものとします。
  2. 当サービスは、加盟店が前項に違反した場合、何らの催告なく本契約を解除できます。この場合、加盟店に損害が生じても当サービスは一切の責任を負いません。

第15条（規約の変更）

  1. 当サービスは、必要に応じて本規約を変更できます。
  2. 変更の30日前までに、管理画面上またはメールにて告知します。
  3. 変更後も本サービスの利用を継続した場合、変更後の規約に同意したものとみなします。

第16条（準拠法・管轄）

  1. 本規約は日本法に準拠します。
  2. 紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
